# MNIST Digit Recognition using Deep Learning

## Overview

This project develops a Convolutional Neural Network (CNN) to recognize handwritten digits using the MNIST dataset. The model is trained to classify grayscale images of digits from **0 to 9** with high accuracy.

## Dataset

* **Dataset:** MNIST Handwritten Digits Dataset
* **Training Images:** 60,000
* **Testing Images:** 10,000
* **Image Size:** 28 × 28 pixels
* **Classes:** Digits 0–9

## Technologies Used

* Python
* NumPy
* Matplotlib
* TensorFlow / Keras
* Scikit-learn
* Jupyter Notebook

## Project Workflow

* Import required libraries
* Load the MNIST dataset
* Explore the dataset
* Preprocess the images
* Visualize sample images
* Build a Convolutional Neural Network (CNN)
* Train the model
* Evaluate model performance
* Predict handwritten digits

## Results

The CNN model achieved high accuracy on the testing dataset, demonstrating its ability to accurately recognize handwritten digits.


## How to Run

1. Clone or download this repository.
2. Install the required libraries:

   ```
   pip install -r requirements.txt
   ```
3. Open the notebook in Jupyter Notebook or Google Colab.
4. Run all the cells in sequence.
